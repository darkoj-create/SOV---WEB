SOV Admin APK 1.3.9.1 — Oružarstvo user-safe stock

- Ordinary users no longer see exact stock quantities in the Oružarstvo catalog.
- User-facing UI shows only Dostupno / Nije dostupno and a request button.
- Admin/Oružar still see exact available/total quantities, inventory progress and warehouse numbers.
- Inventory mode remains Admin/Oružar only.
- Use SUPABASE_ORUZARSTVO_APP_CATALOG_v5_42_USER_SAFE.sql so the app catalog view also masks quantities at SQL level for non-armory users.
