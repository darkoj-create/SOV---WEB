Arhiva nacrta uz svaki objekt.

Canonical modul za nacrte, TopoDroid exporte, Drive linkove i metadata. Članovi vide public nacrte, a Admin/Arhivar mogu dodavati, verificirati i pripremati sync prema objektima i terenskim paketima.

Što radi sada
  • automatski match po imenu objekta i aliasima
  • podržava PDF, slike, TIF/TIFF, ZIP, GPX/KML i TopoDroid fileove
  • sprema lokalni index i, za Admin/Arhivar role, može upisati metadata u Supabase
  • koristi canonical SQL tablicu speleo_object_drawings i audit log

1. Ubaci fileove

Drive arhiv
Odaberi file ili objekt. Ovdje će se prikazati objekt, status matcha, linkovi na kartu/nacrt, georeferenciranje i export za izlet.

2. Georeferenciranje

Za sada unosimo osnovne podatke koji su dovoljni da web pokaže smjer/protegnutost objekta na karti. Kasnije se isti model može spojiti na pravi TopoDroid parser.

Terenski paket
Za izlet kasnije povlačimo objekte + nacrte + TopoDroid + trackove + offline kartu.
