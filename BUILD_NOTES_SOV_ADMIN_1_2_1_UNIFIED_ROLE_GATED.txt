Changes:
1. Unified role preview is now active gating foundation.
   - App still signs in to Supabase from the Home/Tools role card.
   - Synced permissions are stored locally through SovPermissionsStore.

2. Main APK navigation now checks stored permissions before opening modules:
   - Search/Map: allowed when can_view_sov_base or can_view_katastar is true.
   - Izleti: allowed when can_manage_trips is true.
   - Slojevi/Offline: allowed when SOV view or trip management is true.
   - Tools, calculator and Speleo Runner stay available.
   - Home and Settings stay available for blocked/unapproved users.

3. If a role is blocked or not approved:
   - App redirects back to Home.
   - User sees a short toast explaining that the current SOV role has no access.

4. Offline admin fallback remains full-access.
   - This preserves field/admin safety if there is no login session or internet.
   - Once a Supabase session is synced, stored permissions control the gated modules.
