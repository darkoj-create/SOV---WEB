SOV Admin APK 1.2.2 — sync foundation

Version:
- versionCode: 900008
- versionName: 1.2.2-sync-foundation

Što je dodano:
- Role sync foundation za APK.
- Lokalni cache permissiona sada pamti zadnji sync, source, expiry sessiona i zadnju sync grešku.
- App pri otvaranju Postavki pokušava napraviti tihi sync ako postoji spremljena SOV session prijava.
- Sync gumb sada ide preko centralnog SovRoleSyncManagera.
- Ako nema mreže, app jasno koristi spremljeni role cache umjesto da glumi da je sve svježe.
- Ako je session pred istekom ili se ručno klikne Sync, app pokušava refresh token flow.
- Settings kartica prikazuje: rolu, status, online/cached/offline stanje i zadnji sync timestamp.

Namjerno nije dirano:
- Web build.
- SQL schema.
- Oružarstvo UI/modul.
- Katastar/SOV baza logika iz postojećeg builda.

Build napomena:
- Ovo je source ZIP za Android Studio.
- Potpiši istim admin keystoreom ako želiš instalaciju preko postojećeg admin APK-a.
