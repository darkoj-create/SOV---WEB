# SOV Web 5.54 — Izleti view redesign

- `izleti-cloud.html` je redizajniran kao glavni pregled izleta, ne samo unos/sheet.
- Dodan switch **Lista / Kalendar** na istoj stranici.
- Klik na izlet otvara desni detail panel s opisom, metapodacima, dodacima, mail najavom i prognozom.
- Dodavanje novog izleta je desno kao zasebna opcija **+ Dodaj novi izlet**.
- Stari wizard s mapom/točkama nije vraćen; GPX/KML/track/karta ostaju samo opcionalni dodaci.
- Nema novog SQL-a; koristi se postojeći Trips Cloud layer.
