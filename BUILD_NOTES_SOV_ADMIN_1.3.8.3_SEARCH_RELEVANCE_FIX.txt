SOV Admin APK 1.3.8.3 — Oružarstvo search relevance fix

What changed:
- Search no longer returns broad nonsense results for narrow terms like croll/krol/karabiner/pojas.
- Narrow searches are scored against item name, code, category, subcategory and note first.
- Old/global search_text from cache cannot by itself match every row for narrow searches.
- Results are sorted by relevance instead of original catalog order.

Required DB patch:
- Run SUPABASE_ORUZARSTVO_APP_CATALOG_v5_40_2_SEARCH_RELEVANCE_FIX.sql.
- Then refresh Oružarstvo in the APK.

Notes:
- SQL/sync remains based on sov_equipment_app_catalog.
- Izleti and other app modules were not intentionally changed.
